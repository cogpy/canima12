# TODO: Audit Contradiction and Relief-Evidence Schedules

This TODO file records the next refinement tasks generated by the KSM-LEX-DGen-Alexander cycle for the Rule 6(12)(c) reconsideration application.

## Purpose

The reconsideration application is strongest when it remains procedurally focused. The next evidence improvement should therefore be a compact schedule package rather than a broad narrative expansion.

## Schedule 1: Audit Contradiction Schedule

Prepare a table mapping each allegation that originally supported the ex parte order to the later forensic-audit finding or absence of finding. The schedule should include the original allegation, the audit paragraph/page, the applicant’s asserted interpretation, and the relief item supported.

## Schedule 2: Relief-Evidence Matrix

Prepare a one-page matrix mapping each item in the Notice of Motion to the minimum proof anchors required for reconsideration or variation.

## Schedule 3: Bank-Information Request and Refusal Chronology

Prepare a chronological list of requests for bank statements, management accounts, VAT submissions, payroll records, and other financial information, with source emails or letters attached.

## Schedule 4: Entity Role and Governance Register

Prepare a CIPC/entity table for each respondent company, including directors, members, shareholders, mandates, resolutions, and any alleged changes.

## Schedule 5: Transaction and Payment Redirection Schedule

Prepare a transaction-level schedule for alleged transfers, legal-fee payments, customer-payment redirection, creditor defaults, and tax/accounting impacts. This schedule should not assert criminal conclusions unless source bank, ledger, VAT, SARS, or audit exhibits independently support the assertion.
