# Notice Requiring Security for Costs — Rule 47
This template was materialized from the consolidated High Court precedent library so that it can be used directly from the dedicated templates folder.
**Source:** [Uniform Rules Template Library (2026-04-22)](../reference/UNIFORM_RULES_TEMPLATE_LIBRARY_2026-04-22.md)

## Template

```text
NOTICE IN TERMS OF RULE 47

KINDLY TAKE NOTICE THAT the [Plaintiff / Applicant / party requiring security]
requires the [other party] to furnish security for costs in the sum of
[R amount], alternatively in such amount as may be determined by agreement
or by the Court.

TAKE NOTICE FURTHER that such security is required within 10 days after
service hereof, failing which application will be made for an order
compelling the furnishing of security and staying the proceedings pending
compliance.

DATED at [PLACE] on this [DAY] day of [MONTH] [YEAR].

__________________________________
[ATTORNEYS]
TO: THE RECIPIENT PARTY / ATTORNEYS
```
