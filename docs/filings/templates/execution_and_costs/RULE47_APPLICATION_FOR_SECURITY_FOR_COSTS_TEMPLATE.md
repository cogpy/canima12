    # Rule 47 Application for Security for Costs Template

    **Category:** Execution and Costs / Rule 47  
    **Purpose:** Apply for an order compelling provision of security for costs and a stay pending compliance.  
    **When not to use:** Do not launch without first satisfying the procedural prerequisites for the Rule 47 application.

    ## Template

    ```text
    NOTICE OF MOTION: APPLICATION FOR SECURITY FOR COSTS

KINDLY TAKE NOTICE THAT the Applicant applies for an order:

1. Directing the Respondent / Plaintiff / Applicant to furnish security for costs in the sum of [amount], alternatively in such amount as the Court deems fit.
2. Staying the proceedings pending the furnishing of such security.
3. Costs of this application.
4. Further and/or alternative relief.

TAKE NOTICE FURTHER that the affidavit of [Deponent] will be used in support.

DATED at [Place] on [Date].

______________________________
[Attorneys]
TO: THE REGISTRAR
AND TO: ALL PARTIES
    ```
