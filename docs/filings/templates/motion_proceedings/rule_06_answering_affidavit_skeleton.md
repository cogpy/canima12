# Answering Affidavit Skeleton — Rule 6(5)(d)
This template was materialized from the consolidated High Court precedent library so that it can be used directly from the dedicated templates folder.
**Source:** [Uniform Rules Template Library (2026-04-22)](../reference/UNIFORM_RULES_TEMPLATE_LIBRARY_2026-04-22.md)

**Use:** Respondent's substantive answer in application proceedings.  
**When not to use:** Do not use it in place of a plea in action proceedings.

## Template

```text
ANSWERING AFFIDAVIT

I, the undersigned,

[FULL NAME]

do hereby make oath and say:

1. I am the [First / Second] Respondent in this matter.
2. I oppose the relief sought by the Applicant.
3. Save where expressly admitted, every allegation in the founding papers
   inconsistent herewith is denied.

PART A: PRELIMINARY ISSUES
4. At the outset I raise the following preliminary issues:
   4.1 [urgency denied / lack of jurisdiction / non-joinder / authority / etc.]

PART B: RESPONSE TO FACTUAL ALLEGATIONS
5. Ad paragraph 9 thereof:
   5.1 I admit / deny / do not admit the contents.
   5.2 The true position is [detail].
6. Ad paragraph 10 thereof:
   6.1 [response].

PART C: AFFIRMATIVE VERSION
7. The correct chronology is as follows: [detail].

PART D: LEGAL RESPONSE
8. The application falls to be dismissed because [grounds].

PART E: RELIEF
9. I accordingly pray that the application be dismissed with costs,
   alternatively with such order as the Court deems fit.

DEPONENT: __________________
COMMISSIONER OF OATHS: __________________
```
