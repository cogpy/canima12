    # Amicus Curiae Application Template

    **Category:** Appeals and Review / Rule 16A  
    **Purpose:** Seek admission as amicus curiae with supporting affidavit and proposed submissions.  
    **When not to use:** Do not use where the non-party has a direct and substantial interest requiring joinder instead.

    ## Template

    ```text
    NOTICE OF APPLICATION FOR ADMISSION AS AMICUS CURIAE

KINDLY TAKE NOTICE THAT [Name / Entity] applies for admission as amicus curiae in the above matter and seeks an order in the following terms:

1. That the proposed amicus be admitted as amicus curiae.
2. That the proposed amicus be granted leave to file written submissions by [date].
3. That the proposed amicus be granted leave to present oral argument for [time], if so directed.
4. That there be no order as to costs save in the event of opposition.
5. Further and/or alternative relief.

TAKE NOTICE FURTHER that the affidavit of [Deponent] will be used in support hereof.

DATED at [Place] on [Date].

______________________________
[Attorneys / Proposed Amicus]
TO: THE REGISTRAR
AND TO: ALL PARTIES
    ```
