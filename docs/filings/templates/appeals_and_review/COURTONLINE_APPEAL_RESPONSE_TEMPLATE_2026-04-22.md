## Function

This template is not presented as a substitute for full legal advice. It is a **procedural wrapper** designed to help convert the existing respondent-side appellate analysis into a CourtOnline-ready filing note or attorney-drafted notice/submission package. It exists because the generic eight-template set supplied in the attachment does **not** include the document type most closely aligned to the present posture: a disciplined respondent-side opposition layer following the **30 March 2026 notice of application for leave to appeal**.[1] [2]

## Drafting Notes

The wrapper should be kept deliberately disciplined. The current `revstream1` appellate analysis already warns against turning the leave-to-appeal response into a rhetorical overreach document.[1] The best use of the template is therefore to preserve procedural clarity while directing the reader back to the existing documentary contradiction framework.[1] [2]

| Drafting choice | Why it matters |
|---|---|
| Refer expressly to 26 March 2026 and 30 March 2026 | Prevents chronology slippage |
| Avoid re-arguing the entire case in the notice itself | Keeps the filing posture-specific |
| Use "continuation event" language cautiously | Matches the existing repository framework |
| Reserve fuller amplification for affidavit / submissions / heads | Preserves adaptability to court direction |
