    # Rule 41A Mediation Notice Template

    **Category:** Settlement and Mediation / Rule 41A  
    **Purpose:** State a party’s position on referral of the dispute to mediation.  
    **When not to use:** Do not file without addressing whether mediation is agreed, opposed, or conditionally proposed.

    ## Template

    ```text
    NOTICE REGARDING MEDIATION IN TERMS OF RULE 41A

KINDLY TAKE NOTICE THAT the [Plaintiff / Applicant / Defendant / Respondent] [agrees / does not agree] to referral of the dispute to mediation.

The reasons for the party's position are as follows:
1. [Reason 1].
2. [Reason 2].

Without derogating from the party's rights, the following mediator(s) / process is proposed [if applicable]: [detail].

DATED at [Place] on [Date].

______________________________
[Attorneys / Party in person]
TO: ALL OTHER PARTIES / ATTORNEYS
AND TO: THE REGISTRAR, IF REQUIRED
    ```
