# Claim in Reconvention — Rule 24
This template was materialized from the consolidated High Court precedent library so that it can be used directly from the dedicated templates folder.
**Source:** [Uniform Rules Template Library (2026-04-22)](../reference/UNIFORM_RULES_TEMPLATE_LIBRARY_2026-04-22.md)

## Template

```text
CLAIM IN RECONVENTION

The Defendant in convention, now Plaintiff in reconvention, alleges:

1. The parties are as cited in the main action.
2. On [date] [cause of action supporting reconvention].
3. The Plaintiff in convention is indebted / liable to the Defendant in convention for [amount / relief].
4. Despite demand, the said liability remains unsatisfied.

WHEREFORE the Plaintiff in reconvention claims:
(a) [relief];
(b) interest; and
(c) costs.

DATED at [PLACE] on this [DAY] day of [MONTH] [YEAR].

__________________________________
[ATTORNEYS]
```
