# Joinder Application — Rule 10
This template was materialized from the consolidated High Court precedent library so that it can be used directly from the dedicated templates folder.
**Source:** [Uniform Rules Template Library (2026-04-22)](../reference/UNIFORM_RULES_TEMPLATE_LIBRARY_2026-04-22.md)

## Template

```text
NOTICE OF MOTION: APPLICATION FOR JOINDER

KINDLY TAKE NOTICE THAT [APPLICANT FOR JOINDER] applies for an order:

1. Joining [PARTY TO BE JOINED] as the [n-th Respondent / Defendant / Plaintiff / Applicant].
2. Directing that the citation of the parties be amended accordingly.
3. Granting the joined party [time period] to deliver the appropriate process.
4. Costs in the cause, save in the event of opposition.
5. Further and/or alternative relief.

TAKE NOTICE FURTHER that the affidavit of [DEPONENT] will be used in support.

DATED at [PLACE] on this [DAY] day of [MONTH] [YEAR].

__________________________________
[ATTORNEY / PARTY IN PERSON]
TO: THE REGISTRAR
AND TO: ALL EXISTING PARTIES
AND TO: THE PARTY SOUGHT TO BE JOINED
```
