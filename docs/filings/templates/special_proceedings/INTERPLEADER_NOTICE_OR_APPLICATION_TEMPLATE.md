# Interpleader Notice / Application — Rule 58
This template was materialized from the consolidated High Court precedent library so that it can be used directly from the dedicated templates folder.
**Source:** [Uniform Rules Template Library (2026-04-22)](../reference/UNIFORM_RULES_TEMPLATE_LIBRARY_2026-04-22.md)

## Template

```text
INTERPLEADER NOTICE / APPLICATION

KINDLY TAKE NOTICE THAT the Applicant / Sheriff / stakeholder brings this
interpleader in respect of [funds / property / goods] presently held by
[stakeholder], to which competing claims are asserted by:

1. [Claimant 1]; and
2. [Claimant 2].

The Applicant / stakeholder:
(a) claims no beneficial interest in the said property / funds, save for
    charges and costs, if any;
(b) is unable safely to deliver the same without the direction of the Court;
(c) accordingly seeks an order determining the rights of the claimants and
    authorising appropriate delivery / payment.

The Applicant / stakeholder further seeks costs out of the property / funds,
alternatively as directed by the Court.

DATED at [PLACE] on this [DAY] day of [MONTH] [YEAR].

__________________________________
[ATTORNEYS / SHERIFF / STAKEHOLDER]
TO: THE REGISTRAR
AND TO: ALL CLAIMANTS
```
