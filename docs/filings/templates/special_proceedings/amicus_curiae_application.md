# Amicus Curiae Application — Rule 16A / Inherent Jurisdiction
This template was materialized from the consolidated High Court precedent library so that it can be used directly from the dedicated templates folder.
**Source:** [Uniform Rules Template Library (2026-04-22)](../reference/UNIFORM_RULES_TEMPLATE_LIBRARY_2026-04-22.md)

## Template

```text
NOTICE OF APPLICATION FOR ADMISSION AS AMICUS CURIAE

KINDLY TAKE NOTICE THAT [NAME / ENTITY] applies for admission as amicus
curiae in the above matter and seeks an order in the following terms:

1. That the proposed amicus be admitted as amicus curiae.
2. That the proposed amicus be granted leave to file written submissions by [DATE].
3. That the proposed amicus be granted leave to present oral argument for [TIME], if so directed.
4. That there be no order as to costs save in the event of opposition.
5. Further and/or alternative relief.

TAKE NOTICE FURTHER that the affidavit of [DEPONENT] will be used in support hereof.

DATED at [PLACE] on this [DAY] day of [MONTH] [YEAR].

__________________________________
[ATTORNEYS / PROPOSED AMICUS]
TO: THE REGISTRAR
AND TO: ALL PARTIES
```
