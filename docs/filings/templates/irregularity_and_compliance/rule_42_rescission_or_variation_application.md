# Rule 42 Rescission / Variation Application
This template was materialized from the consolidated High Court precedent library so that it can be used directly from the dedicated templates folder.
**Source:** [Uniform Rules Template Library (2026-04-22)](../reference/UNIFORM_RULES_TEMPLATE_LIBRARY_2026-04-22.md)

## Template

```text
NOTICE OF MOTION: APPLICATION IN TERMS OF RULE 42

KINDLY TAKE NOTICE THAT the Applicant applies for an order:

1. Rescinding / varying the order granted on [date] under case number [number].
2. Declaring that the said order was erroneously sought and/or erroneously granted,
   alternatively that it resulted from a mistake common to the parties,
   alternatively varying it to the extent set out below.
3. Costs of this application [as appropriate].
4. Further and/or alternative relief.

TAKE NOTICE FURTHER that the affidavit of [DEPONENT] will be used in support.

DATED at [PLACE] on this [DAY] day of [MONTH] [YEAR].

__________________________________
[ATTORNEYS]
TO: THE REGISTRAR
AND TO: ALL AFFECTED PARTIES
```
