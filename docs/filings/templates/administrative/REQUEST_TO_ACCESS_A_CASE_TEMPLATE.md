    # Request to Access a Case Template

    **Category:** Administrative / CourtOnline  
    **Purpose:** Request electronic access to an existing court file after proper appointment, joinder, or authorised participation.  
    **When not to use:** Do not use as a substitute for a substantive notice of opposition, defence, or application.

    ## Template

    ```text
    REQUEST TO ACCESS A CASE

Court / Platform: [CourtOnline / e-Justice]
Case Number: [YYYY-NNNNNN]
Division: [Division]
Case Title: [Case title]

Requesting Party:
Name: [Full name]
Capacity: [Attorney / Party / Candidate attorney / Correspondent]
Firm: [Firm name if applicable]
Email: [Email]
Mobile: [Number]
Practitioner Number / ID: [Number]

Basis for Access Request:
1. I act for / am the [party] in this matter.
2. My authority appears from [notice of appointment / mandate / joinder order / written authority] attached hereto.
3. I request access to the electronic file to receive notices, inspect filed documents, and upload authorised process.

Documents Attached:
- Notice of appointment / mandate / court order
- ID / practitioner card / firm letterhead [as applicable]
- Any platform-specific access form

Requested Outcome:
That access be granted to the above case and linked to the requesting practitioner / party profile.

Signed at [Place] on [Date].

______________________________
[Name / Signature]
    ```
