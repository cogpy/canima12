# Request to Access a Case

This template provides the administrative wrapper for obtaining CourtOnline access after appointment, joinder, or other procedural recognition.

## Template

```text
REQUEST TO ACCESS A CASE

Case Number: [YYYY-NNNNNN]
Court / Division: [DIVISION]
Applicant / Plaintiff: [NAME]
Respondent / Defendant: [NAME]

I, [FULL NAME], in my capacity as [attorney of record / counsel / party /
authorised representative], hereby request access to the electronic case
file for the above matter on CourtOnline / the e-justice system.

The basis for this request is as follows:
1. I have been appointed as [role] for [party] in terms of the attached
   notice / mandate / order dated [DATE].
2. I require access for purposes of filing, receiving service, inspecting
   the record, and managing the proceedings electronically.
3. My practitioner / identity details are as follows:
   3.1 Firm / organisation: [DETAIL]
   3.2 Practitioner number / ID number: [DETAIL]
   3.3 Email address linked to the platform: [DETAIL]
   3.4 Telephone number: [DETAIL]

Attached in support of this request are:
(a) [notice of appointment / mandate / order / power of attorney];
(b) [identity / practitioner proof, if required]; and
(c) any further supporting documents required by the platform or Registrar.

DATED at [PLACE] on this [DAY] day of [MONTH] [YEAR].

__________________________________
[NAME]
[ROLE]
[EMAIL]
[PHONE]
```
