## Important note

The selected source is the 49-page Rule 6(12)(c) reconsideration PDF. The 53-page Court Online copy is retained as the secondary reference. Both source copies show the same late-document internal page-order anomaly: internal affidavit page 41 appears before internal page 40, followed by internal page 42. The transcription preserves the PDF source order rather than silently reordering the document.
